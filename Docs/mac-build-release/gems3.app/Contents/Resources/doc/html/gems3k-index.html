The documentation of standalone GEMS3K code consists of the above-listed HTML files that can be reached via this file "Docs/html/index.html" in the trunk GEMS3K subversion repository. The source code documentation in doxygen style can be accessed via "Docs/doxy/html/index.html".  

The recommended way to prepare GEMS3K input files consists in creating a modeling project in GEM-Selektor package, creating and calculating chemical systems (equilibria), and then exporting the GEM task records to GEMS3K I/O files.
