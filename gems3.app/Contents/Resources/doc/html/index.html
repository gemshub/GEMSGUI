GEM-Selektor version 2-PSI 

ON-LINE HTML DOCUMENTATION

(still under construction)

HTML files listed below comprise an actually updated version of the files located in the  /program/doc/html directory of the current GEMS distribution. These files should be readable in any HTML browser and can be displayed at runtime also in the GEM-Selektor Helpviewer  browser  by clicking on "Help" buttons, icons or  F1  keystroke.

Users that have a fast internet connection (e.g. LAN, DSL, cable) can access these files directly from GEMS Helpviewer (after switching to the internet link in the "Preferences" dialog). Please, note that this remote variant of help viewing does not work stable under some Linux distributions. It is also known that under Win32, the helpviewer may freese after switching back to the locally installed help files.

Attention! In this version (2.1.0-PSI), not all HTML help files are completed; those to be provided in version 2.2 or later are listed separately.
