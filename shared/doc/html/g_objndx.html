<>This is  OBJNDX.HTML  file used internally by GEM-Selektor runtime help system.
Entries in this file redirect the help system to display help pages prepared in various HTML files.


Rules for setting up DOD keywords and links to html files

Example:   <a href="gm_dcomp.html#TCint_0">TCint[0]</a> Comment

The entry must contain a HTML code like this:
     
              <a href="html_file_namel#label">keyword</a>
Here:        html_file_name          is the name of html file where to jump;
             label                          position label (anchor) in this  file              keyword                   GEM data object label, optionally with indexes.

- if dimension N  in the DOD exceeds 1 and dimension M in the DOD exceeds 1:             keyword[indexN][indexM]        locates a reference to the help page only for this cell      keyword[][indexM]                   locates a reference to the help page for any cell in the column indexM            
keyword[indexN]                      locates a reference to the help page for any cell in the row indexN            
keyword                                    locates a  reference to the same help page for any cell  -  if only one dimension (N or M)  in DOD exceeds 1 :             
keyword[index]                        locates a reference to the help page only for a cell with this index              
keyword                                  locates a  reference to the same help page for any cell
- if both dimensions N and M are set to 1, or the DOD refers to text:
keyword                                  locates a  reference to the help page 

All links for the same keyword must be located together in this file (i.e. the search stops when
another keyword is found).
