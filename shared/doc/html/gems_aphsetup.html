Automatic Setup of Aqueous and Gas/Fluid Phases

This dialog facilitates setup of definitions of the aqueous electrolyte phase and the gas/fluid mixture phase that will be automatically assembled when opening (or creating) a GEMS modelling project. For the underlying models and equations, see the theorgem.html manual and pdf manual files linked to it.

The first tab "Select Aqueous Electrolyte Model" lets you define a built-in function for calculating activity coefficients of individual aqueous species. These can be selected by clicking on radio buttons on the left side, or pressing  Alt-D, Alt-H, Alt-3, ... key combinations. The two last choices should be used when none of the built-in functions is applicable, or no aqueous phase should be included into the chemical system definition. In the "Parameters for the aqueous model" frame, you can change some default parameters used in some of the built-in aqueous activity functions which are described below.

Most of the aqueous activity coefficient models that are currently supported by GEM-Selektor code belong to the class of extended Debye-Hückel (EDH) models (see document Activity-Coeffs.pdf for details about the underlying theory and references). In these models, the extended term (b_gamma) can be a function of pressure and temperature or a constant value. In the select box "b_gamma(P,T) mode" the operation mode of the b_gamma function needs to be selceted. If the mode "constant" is chosen, the value of b_gamma at reference state conditions (25 deg. C, 1 bar) needs to be entered in the "b_gamma(1,298) value" field above the select box.

Alternatively, for calculations at hydrothermal conditions, the b_gamma term can be a function of temperature and pressure. In this case, four b_gamma functions for different so-called background electrolytes (NaCl, KCl, NaOH, KOH) can be selected. The concept of the background electrolyte means that it is assumed that the aqueous solution that is modeled is dominantly composed of that particular electrolyte and the model should be chosen accordingly. If in your project you mainly deal with NaCl-rich fluids (the most common case), you should then choose the NaCl model. Note that the T-P-dependent b_gamma functions should be used in conjunction with the extended Debeye-Hückel model after Helgeson and coworkers (Oelkers and Helgeson 1990; Pokrovski and Helgeson 1995; 1997a; 1997b), the second model in the list which has the code letter "H", because of thermodynamic consistency. If a T-P dependent b_gamma model is selected, there is no need to enter a value into the "b_gamma(1,298) value" field, because this value is automatically calculated from a built-in routine. However, it is neccessary to set up the appropriate value of the ion size parameter for each of the four background electrolytes. The value of the ion size parameter (also called distance of closest approach) is entered in the "Common a0 value" field just below the model select box. From the Debye-Hueckel models described above the activity coefficients of aqueous electrolyte species (charged ions and complexes) will be calculated. For uncharged species (neutral ion pairs and complexes, neutral dissolved gas molecules such as CO2, H2S etc.) and for water solvent different activity coefficient models will be used. Four options for these models exist, which are specified by a number code that is entered in the "Policy_to_neutral:" field. Note that one should select the basic model (click on the appropriate radio button on the left side) before setting up details of the calculation mode and parameters (T-P-dependence of parameters, calculation of activity coefficients for neutral species and water solvent). Because of thermodynamic consistency, those options (from the table above) that explicitly calculate the activity coefficient of water solvent should only be used with the extended Debeye-Hückel model after Helgeson and coworkers.

Clicking on the "Check" button displays a record key of the Phase record to be automatically generated;  the chosen model and its parameters will be identified in the PhName and PhNote fields. You can check them later when you open the "Phase" page of the "System" window from the "Single Thermodynamic System" dialog, mark the phase record key and press F7. Note that, when the same modelling project is opened next time, this Phase definition record will be automatically deleted and generated again without displaying this setup dialog.

The second tab "Select Gas/Fluid Mixture Model" defines the automatically generated Phase record for gas or fluid mixture. It functions in a way similar to that for the aqueous phase. The generic models that are currently supported are (1) an ideal gas mixture, and (2) non-ideal gas (fluid) mixture calculated with the Churakov-Gottschalk (CG) model. The Peng-Robinson-Stryjek-Vera (PRSV) model for non-ideal gas (fluid) mixtures that is also implemented in the GEM-Selektor code requires that first a specific Phase record (in database management mode) is created. The gas species that are included in the gas mixture need to be selected and binary interaction parameters can be entered into the ph_cf table. If a phase record is prepared in this way, it can then be selected in the Setup of Aqueous and Gas/Fluid Phase wizard by chosing the option "Do not generate; select a user-defined Phase record from database,U". After selecting this option, a list of phase records that are available in the database will be displayed. Note that no additional parameters for the gas/fluid mixture (right side of the wizard) are used in this version of GEM-Selektor.

The setup of automatically generated Phase records for aqueous and gas phases can be changed any time upon opening this modelling project. To do this, uncheck the "Retain setup of aqueous (and gas) phases" option in the "Modelling Projects" dialog before selecting in the list an exitsting project to be opened.
