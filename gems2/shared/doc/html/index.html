GEM-Selektor version 2-PSI 

ON-LINE HTML DOCUMENTATION

(still under construction)

HTML files listed below comprise an actually updated version of the files located in the  /program/doc/html directory of the current GEMS distribution. These files should be readable in any HTML browser and can be displayed at runtime also in the GEM-Selektor Helpviewer  browser  by clicking on "Help" buttons, icons or  F1  keystroke. Users that have a fast a internet connection (e.g. LAN, DSL, cable) can access these files directly from GEMS Helpviewer (after switching to the internet link in the "Preferences" dialog).

Attention! In this version (2.0.0-PSI), not all HTML files are completed yet;
those planned to be ready in version 2.1 or later are listed separately.
