(1) This Main Window appears after you have started the GEM-Selektor executable file:


 

In this screen image, the program is in the Database Mode. Using toolbar buttons in the upper left corner, you can switch between two main modes: Database Mode and EqCalc (Equilibrium Calculations) Mode. By clicking buttons (icons) on the left pane, you can open the respective module windows (e.g. DComp). All toolbar buttons show tooltips (hover the cursor over the button for 1-2 s).

(2) From MainDialog, the "Help Contents" button (with question mark) opens the Help Viewer - a simple browser that shows the HTML documentation. Help Viewer can also be open from the "Help" menu; the help pages are also shown by pressing F1 on any data object on any GEM-Selektor screen forms.


If you open Help Viewer and see no content then close the Help Viewer window, click "Help" "Preferences" in the Main Window menu, and in the appearing Preferences Dialog check and edit the path to help files.

(3) If you read this screenshot tutorial and run GEM-Selektor in parallel then switch to the Equilibria Calculation Mode (and proceed to the next page of Screenshot Gallery).
