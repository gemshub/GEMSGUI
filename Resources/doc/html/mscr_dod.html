DOD lists (for Math Scripts)

8. List of Data Object Labels (usable as Math Script Variables)

Data Object Descriptors (DOD): In GEMS, the DOD array makes a bridge between the users interface, the internal data structures, the database records and the numerical subroutines. Each data object referred via DOD has a label which appears usually appears on screen forms and serves as a link to the on-line help system. The DOD labels with indices also serve as variables in Math Script expressions. DOD also contains information on the current dimensions and type of the data.  In the lists below, only a subset of the full list of numerical data object labels are provided. The objects most frequently used in math scripts are highlighted in red.

8.1. Data objects for non-ideality expressions in phase definitions only

The data object descriptors listed below are automatically updated by GammaCalc() subroutine for each multicomponent phase. They can be addressed from in the Math Script expressions as variables by names (for scalars) or by names and indices (for vectors and matrices) using  js  and is  indices which are applicable within this phase definition only.  Data cell types: I_  integer;  F_ float;  D_ double.  js  index is automatically updated by the program in "public" mode of calculation of activity coefficients.
